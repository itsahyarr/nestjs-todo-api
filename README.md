# Todo API

## Installation

```bash
# install dependencies
$ bun install
```

## Running the app

```bash
# development
$ bun start

# watch mode
$ bun start:dev

# production mode
$ bun start:prod
```

## Test

```bash
# unit tests
$ bun test

# e2e tests
$ bun test:e2e

# test coverage
$ bun test:cov
```
